# The Last Meadow Clicker Bot

A simple JavaScript-based clicker bot for The Last Meadow, a popular discord game.

## Features
* Automated clicking for The Last Meadow
* Easy to use and configure
* No installation required, just copy and paste the code
* If you reduce the speed, you will get an error. 26 is the maximum speed at which the game will not give an error.
